AI STUDIO – TEXT WIREFRAMES
===========================

GLOBAL LAYOUT
-------------
- Left Sidebar:
  - App logo/title
  - Vertical nav: Script / Audio / Image / Video / Settings
- Main Content:
  - Page header (title + short description)
  - Main body (forms, tables, grids)
- Right Library Panel:
  - Title "Library"
  - Filter dropdown
  - Scrollable list of assets (compact cards or rows)

SCRIPT SCREEN (/script)
-----------------------
Header:
  - Title: 스크립트
  - Subtitle: “주제와 옵션을 입력하면 한 호흡 단위 스크립트를 자동 생성합니다.”

Body:
  [Left column – Script Form + Scene List]
    Section: Script Options
      - Row 1: Target platform (select), Category (select)
      - Row 2: Tone/Style (select), Target length (select)
      - Toggles: include hook, include CTA, etc.
    Section: Prompt
      - Label “프롬프트”
      - Multiline textarea
      - Helper text with example
    Section: Actions
      - Primary button: [스크립트 생성]
    Section: Scene List
      - For each scene:
        - Index (01, 02, 03…)
        - Scene text (2–3 lines)
        - Icon buttons: Edit, Delete, Copy, Go to Audio, Go to Image
      - Button “전체 스크립트 보기” → opens modal.

  [Right column – Library]
    - Dropdown filter (All / Recent)
    - List of projects (title, scenes count, created date)

Modal: Script Detail
  - Title bar: project name
  - Scrollable area with full script text and scene separators
  - Bottom buttons: Copy all / Close

AUDIO SCREEN (/audio)
---------------------
Header:
  - Title: 오디오
  - Subtitle: “생성된 스크립트로 TTS 오디오를 만듭니다.”

Body:
  [Left – Script selection + table]
    Section: Script Selector
      - Label “스크립트 선택”
      - Dropdown listing script projects
    Section: Scene Audio Table
      - Columns:
        - # (scene index)
        - Scene preview text
        - Voice (select)
        - Speed (slider or select)
        - Status badge (Not generated / Pending / Ready / Error)
        - Actions:
          - [TTS 생성] button
          - Play / Stop
          - Regenerate
          - Delete
      - Top-right: “전체 TTS 일괄 생성” button

  [Right – Library]
    - List of audio groups by project

Modal: Audio Detail
  - Scene text
  - Audio player with timeline
  - Info (voice, speed, length)
  - Actions: Download, Regenerate, Close

IMAGE SCREEN (/image)
---------------------
Header:
  - Title: 이미지
  - Subtitle: “장면별 이미지 생성 및 관리”

Body:
  [Left – filters + scene cards]
    Section: Filters
      - Project selector
      - Status filter (All / Missing / Generated)
    Section: Scene Grid
      - 2~3 columns of cards
      - Card:
        - Top: scene index & short label
        - Middle:
          - Thumbnail if exists, otherwise placeholder
        - Bottom:
          - Scene text (2 lines)
          - Buttons: [이미지 생성], Open detail, Send to Video

  [Right – Library]
    - Thumbnails grouped by project

Modal: Image Generation
  - Left: scene text, base prompt
  - Right: style prompt input, model options, size, number of variations
  - Button: [Generate]
  - Result thumbnails with “set as main” selection

VIDEO SCREEN (/video)
---------------------
Header:
  - Title: 비디오
  - Subtitle: “이미지로 짧은 영상 생성”

Body:
  [Left – selection + options]
    Section: Source Selection
      - Radio buttons: Use scenes / Use selected images
      - Image grid with checkboxes
    Section: Video Options
      - Aspect ratio (select)
      - Duration (input)
      - Motion style (select)
      - Optional BGM selection
      - Button: [비디오 생성]

  [Right – results]
    - Video cards showing thumbnail, title, duration, status
    - Icon buttons: Play (modal), Download, Delete

Modal: Video Detail
  - Video player
  - Meta information (resolution, duration, images used)
  - Buttons: Re-generate with same options, Close

SETTINGS SCREEN (/settings)
---------------------------
Tabbed layout:
- Models tab:
  - Script model select
  - Image model select
  - TTS provider select
  - Video provider select
  - API key inputs + “Test” buttons
- Defaults tab:
  - Default language
  - Default tone
  - Default image style
  - Default TTS voice
- System tab:
  - Theme toggle (light/dark)
  - Auto-save interval
  - Export / Import configuration
