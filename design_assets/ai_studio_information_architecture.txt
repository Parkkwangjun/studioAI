AI STUDIO – INFORMATION ARCHITECTURE (IA)
========================================

Top-Level Layout
----------------
Application Root
- Global Layout
  - Left Navigation Bar (LNB)
  - Main Content Area
  - Right Library Panel
  - Global Modals & Toasts

Left Navigation Items
---------------------
- /script    → Script section (스크립트)
- /audio     → Audio section (오디오)
- /image     → Image section (이미지)
- /video     → Video section (비디오)
- /settings  → Settings (설정)

1. Script Section (/script)
---------------------------
Pages / States:
- Script Home
  - Script generator form
  - Scene list output
- Script Detail Modal
  - Full script view (all scenes)
Right Library:
- Project list (scripts)

2. Audio Section (/audio)
-------------------------
Pages / States:
- Audio Home
  - Script selector (“Select Script File”)
  - Scene audio table (one row per scene)
- Audio Detail Modal
  - Single-scene audio preview
Right Library:
- Tabs: Scripts / Audios (optional)
- Audio groups by project

3. Image Section (/image)
-------------------------
Pages / States:
- Image Home
  - Filters (project, status)
  - Scene cards in a grid (scene text + thumbnails + action buttons)
- Image Generation Modal
  - Scene text + prompt editing + model options + results
Right Library:
- Image thumbnails grouped by project

4. Video Section (/video)
-------------------------
Pages / States:
- Video Home
  - Source selection (scenes / images)
  - Image selection grid
  - Video options form
  - Video result card list
- Video Detail Modal
  - Video player + metadata
Right Library:
- Video list with thumbnails

5. Settings Section (/settings)
-------------------------------
Sub-sections (tabs inside Settings):
- Models
- Defaults
- System

IA Summary
----------
- All features are accessible by left navigation.
- Each major section has a main workspace + right library.
- Modals are used for detail views (full script, audio preview, image generation, video preview).
